Welcome to Project Eden

The project aims to create a structured platform for daily diary entries, skill tracking, rank requirements, and more, replacing traditional notebooks. The theme is inspired by my favorite Pokémon, with Litten representing someone dear to me.

Goal for the web application

  • [PsyDuck]: Able to Create, delete, update, view Diary entry
  • [Chansey]: Able to Create, delete, update, view Tasks
  • [Snorlax]: Able to Create, delete, update, view Daily Routine
  • [Disguise]: Able to Select skills, level them up based on tasks completed
  • [Machamp]: Able to view rank and see what requirement needed to be complted to meet ranking up
  • [Litten]: Able to view profile with statistics

Rank Requirements

The purpose of our demanding requirements is to show that you're committed to self-improvement in various areas—from intelligence to religion to skills. This rank system proves your ability to excel in diverse aspects of life, showcasing your dedication and capability for growth.
